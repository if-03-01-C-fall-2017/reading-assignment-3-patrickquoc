HTML Lists

An unordered list starts with the <​ul> tag. Each list item starts with the <​li> tag. Example:
<​ul>
<​li>Example<​/li>

 An ordered list starts with the <​ol> tag instead of the <​ul> tag. Example:

  1. This is an ordered list.
  2. Ordered list are numbered.

HTML also has description lists. The <​dl> tag defines the description list, the <​dt> tag defines the term (name), and the <​dd> tag describes each term.

CSS Lists

CSS can set different list item markers for lists. It is possible to set a picture as a list markers. You can set a background color for lists too.
With list-style-type you can choose implemented list style item.
Setting images as a list item marker works with list-style-image. (Example: list-style-image: url('sqpurple.gif'))
You can change the list item marker position with list-style-position.(Example: list-style-position: inside;))
The list-style-type:none property can also be used to remove the markers/bullets.

HTML Tables

An HTML table is defined with the <​table> tag. ach table row is defined with the <​tr> tag. A table header is defined with the <​th> tag. A table cell is defined with the <​td> tag. Example:

Continent Country Capital
Europe Austria Vienna
North America United States of America Washington, D.C.
Asia South Korea Seoul

You can set the row or column span by using for example<​th colspan="2">Header<​/th>. This will look like this:

Name Grade
Pooh Bear D C+

You can use the <​caption> tag to add a caption to a table.

CSS Tables

You can specify table borders in CSS, by using the border property. In CSS this will look like this: border: 1px solid black;
The border-collapse property sets whether the table borders should be collapsed into a single border. Example: border-collapse: collapse;
Width and height of a table are defined by the width and height properties. Example: height = 80px
For striped tables, use the nth-child() selector and add a background-color to all even or odd table rows. Example: tr:nth-child(even){background-color: #ffffff}

CSS Borders

CSS can define which border style you want. These are the following border styles:

  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border
  • groove - Defines a 3D grooved border. The effect depends on the border-color value
  • ridge - Defines a 3D ridged border. The effect depends on the border-color value
  • inset - Defines a 3D inset border. The effect depends on the border-color value
  • outset - Defines a 3D outset border. The effect depends on the border-color value
  • none - Defines no border
  • hidden - Defines a hidden border

You can change the border width, by using border-width. This is how it looks in CSS: border-width: 7px;
The border-color property sets the color of the border. Example: border-color = #ffffff;
It is possible to change the border style for each side. Each border are defined for instance like this: border-bottom-style: dotted; This works for bottom, top, left and right.
To shorten your CSS code you can define the borders with only one property. This will look like this: border: 5px solid red;. The first border property is the width, the second one the style (which is required) and the last one the color.
You can even make the borders round, by doing this: border-radius: 5px;
